## Requirements

### yaml
please install and build yaml.
```zsh
git clone https://github.com/jbeder/yaml-cpp.git
cd yaml-cpp
mkdir build
cd build
cmake ..
make
sudo make install
```

### GUI
please do this.

```zsh
echo 'export PATH=$PATH:/usr/local/share/dotnet' >> ~/.zshrc
source ~/.zshrc
sudo dotnet workload install maui
```

### others
```zsh
pip install -r requirements.txt
```
